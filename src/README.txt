SNAKE GAME by Joni Toikkanen

- Open the main.py file and run it

- Move the snake with arrow keys

- You will have 5 game runs for each game session

- Everytime snake eats the food, you will receive 10 points

- Game session ends when the snake hits the screen or itself

- Game closes automatically when you have played all the 5 game runs

- Have a great game sessions!
